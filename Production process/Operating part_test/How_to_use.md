# 프로젝트 실행 가이드

## 1. 가상환경 세팅
`torch` 버전 충돌을 막기 위해 **2개의 가상환경**을 만들어주세요.
* **환경 A:** AI Agent용
* **환경 B:** TTS Server용

## 2. 파일 설정
* **`AI_Agent.py`**: 본인의 API 키 입력
* **`tts_engine.py`**: 원하는 목소리 파일 경로 입력 (없을 경우 `TTS` 폴더 내 파일 사용)

## 3. 실행 방법
각각 분리된 환경(터미널 2개)에서 아래 명령어를 실행합니다.

**터미널 1 (환경 B 활성화):**
> python tts_server.py

**터미널 2 (환경 A 활성화):**
> python main.py
